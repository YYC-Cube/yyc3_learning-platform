# Focus SaaS 平台系统操作指南 (草案)

## 1. 核心流程走查

### 1.1 发现与探索

- **仪表盘**: 登录后默认进入仪表盘，查看热门模块和最近学习进度。
- **分类导航**: 使用左侧导航栏快速切换 IA、SEO、E-commerce 等不同领域的课程。

### 1.2 解锁与支付

- **Premium 按钮**: 点击“解锁 - 250€”进入支付模拟流程。系统将统一视觉风格，强化高端商业感。
- **即时访问**: 解锁后，模块状态从“待解锁”变为“已解锁”，用户可立即点击“开始学习”。

### 1.3 课程学习

- **视频播放器**: 支持多课时切换、自动保存进度及全屏模式。
- **互动社区**: 在学习过程中，可随时进入“社区讨论”频道与全球精英交流。

### 1.4 结业与认证

- **项目看板**: 提交实战项目（Kanban 模式管理），P0 优先级项目优先审核。
- **获取证书**: 模块 100% 完成后，系统自动生成带防伪水印的高级证书。

## 2. 响应式与交互指南

- **折叠屏适配**: 系统自动感应屏幕状态，在展开态显示双栏/多栏布局，折叠态自动隐藏侧边栏并启用汉堡菜单。
- **双语切换**: 顶部 Header 包含语言切换按钮，支持中法文一键无缝转换。

## 3. 资产与技术支持

- **图标系统**: 全站采用 Lucide-React 图标库。
- **动态资产**: 所有 SVG 和 figma:asset 均已针对 Vercel/生产环境进行路径优化。
