# YYC3-Learning-Platform 系统架构与本地化同步文档 (2026-02-03)

## 1. 核心定位

YYC3 是一款针对中国市场深度定制的商业培训 SaaS 平台。视觉风格采用“蓝色旋变”（Blue Swirl）深海蓝主题，强调极简、高端、商务感。

## 2. 技术栈同步

- **前端**: React + Tailwind CSS v4 + Framer Motion
- **后端**: Supabase Edge Functions (Deno 运行时)
- **数据持久化**: Supabase KV 存储 (kv_store_dae9c128)
- **多语言**: 自研 LanguageContext，默认锁定 `zh` (中文环境)

## 3. 本地化增强 (Localization Sync)

### 3.1 中文搜索优化

- **模糊匹配**: 在 `App.tsx` 中集成了 `useMemo` 驱动的中文模糊匹配逻辑。
- **范围**: 覆盖课程标题、副标题及分类标签。

### 3.2 移动端排版优化 (Kerning & Spacing)

- **字体栈**: 优化了 `globals.css` 中的字体回退顺序（PingFang SC -> Microsoft YaHei）。
- **间距调整**:
  - 针对中文汉字密度，在移动端增加了 `tracking` (0.02em) 以提升阅读舒适度。
  - 标题行高锁定为 `1.25`，正文行高锁定为 `1.75`。

## 4. 功能闭环状态 (Interaction Update)

- [x] **图表渲染修复 (Chart Rendering Fix)**:
  - 修复了 Recharts `ResponsiveContainer` 在初始化时因容器尺寸未计算完成导致的 `width/height (-1)` 运行时错误。
  - 为所有图表容器强制指定了显式高度及 `min-w-0` 约束，确保在 Flex 布局及动画切换过程中尺寸计算的稳定性。
- [x] **仪表盘模块升级 (Advanced Dashboard)**:

- [x] **智能多级路径预测 (Multi-level Path Prediction)**:

- [x] **全场景 Breadcrumbs 覆盖**:

- [x] **移动端面包屑适配**: 增加自适应宽度与平滑滚动。
