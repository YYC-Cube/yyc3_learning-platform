# export-kv-cli.js

## 🚀 使用方法

1. 保存为 `export-kv-cli.js`。
2. 安装依赖：

   ```bash
   npm install @supabase/supabase-js
   ```

3. 运行脚本：
   - 导出全部数据

     ```bash
     node export-kv-cli.js
     ```

   - 导出某个用户的数据

     ```bash
     node export-kv-cli.js --user abc123
     ```

   - 导出某个前缀的数据

     ```bash
     node export-kv-cli.js --prefix song:
     ```

   - 限制导出行数（例如 200 行）

     ```bash
     node export-kv-cli.js --limit 200
     ```

---

这样你就能在命令行里一步到位选择导出范围，生成 JSON 文件。
